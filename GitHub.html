Comenzi GitHub

1. git init

Inițializează un nou repository Git local.

git init

2. git clone

Clonează un repository existent de pe GitHub.

git clone https://github.com/user/repo.git

3. git add

Adaugă fișiere în zona de staging.

git add nume_fisier

4. git commit

Salvează modificările din zona de staging în istoria repository-ului.

git commit -m "Mesaj de commit"

5. git push

Trimite modificările locale către un repository remote.

git push origin branch_name

6. git pull

Obține și îmbină modificările dintr-un repository remote în repository-ul local.

git pull origin branch_name

7. git status

Arată starea curentă a repository-ului.

git status

8. git branch

Listează, creează sau șterge branch-uri.

git branch
